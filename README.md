Adventures in Clonk
===================
A RPG made using BYOND (Build Your Own Net Dream), WIP project.

To develop, go open the CliffieRPG.dme in the /Clonk folder in Dream Maker.  Note that running the project will end up showing a "guest" user in the stats, but that's okay.

This is GPLed and all assets so far are either from tutorials or by me.
